# 롤랩

웹 브라우저에서 사용하는 피아노 롤 편집기입니다. FL Studio 스타일의 노트 입력, 트랙 관리, 스케일 표시, 벨로시티 편집, JSON 저장과 불러오기를 제공합니다.

## 실행

```bash
npm install
npm run dev
```

## 배포

`main` 브랜치에 푸시하면 GitHub Actions가 빌드 결과를 GitHub Pages로 배포합니다.

## 주요 기능

- 노트 그리기, 선택, 이동, 길이 조절
- 트랙 추가, 음소거, 솔로, 삭제
- 템포, 박자, 마디, 스냅 설정
- 스케일 루트와 모드 선택
- 벨로시티 편집
- 프로젝트 JSON 저장과 불러오기
